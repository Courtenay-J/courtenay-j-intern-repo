# React + Vite

This React project was built using Vite, via W3Schools's [React Get Started](https://www.w3schools.com/react/react_getstarted.asp) guide. It was then modified to include Tailwind, via the [Install Tailwind CSS](https://v3.tailwindcss.com/docs/guides/create-react-app) guide.

It does not currently use the React Compiler, though it does have [`typescript-eslint`](https://typescript-eslint.io) enabled.
