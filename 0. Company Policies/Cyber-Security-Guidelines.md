# Cyber Security Guidelines
## Reflection
### What security measures do you currently follow, and where can you improve?
**Currently following:**
- Using strong and unique passwords, and separate emails for work, university, and recreation
- Subscribed to services that inform me of data breaches (e.g. haveibeenpwnd)
- My email clients always show the full email address of the sender, making it easier to spot phishing attempts
- Read up on common phishing scams, including platform-specific ones (e.g. the recent discord "age verification" scam)
- All my devices have complicated pins (not my birthdate!) and quick screen timeouts

**Room for improvement:**
- I have been dragging my feet on 2FA
- I do not currently use a password manager
- Lacking experience in checking the security of packages (Scanning with antivirus does not always catch a malicious script)

### How can you make secure behaviour a habit rather than an afterthought?
- *Always* check where a message is coming from - a quick glance is not enough
- *Always* ask if the behaviour makes sense. 
- *Never* install things in a hurry. Check the source, check the number of versions, the number of downloads, etc.

### What steps will you take to ensure your passwords and accounts are secure?
- Enable 2FA on all work accounts, and use a password manager.
- Only log in to work accounts on personal devices, which are password locked and screenlock quickly
- Never give login information to people via email, discord, or elsewhere
- Always check the address of incoming emails/text messages to avoid phishing, and check w/ Focus Bear if something looks suspicious
- Never leave sensitive information in an unsecure location (e.g. unlocked device, unencrypted usb)

### What would you do if you suspected a security breach or suspicious activity on your account?
- Change password and force a logout of all sessions (if possible on that service)
- Change password on any other service that uses the same email-password combo
- Scan computer for malware
- Contact Focus Bear about the breach and explain what may have been affected
